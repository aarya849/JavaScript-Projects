# JavaScript Projects

This repository contains basic JavaScript projects built using HTML, CSS, and JavaScript.
These projects focus on DOM manipulation, event handling, and user interaction.

## Projects Included

### 1. Reminder App

A simple reminder application where users can add and manage reminders.
It helps understand form handling, event listeners, and dynamic DOM updates.

**Features:**

- Add reminders
- Display reminders dynamically
- Simple and clean UI

### 2. Search Bar

A search bar project that filters content based on user input.
This project focuses on real-time search and array filtering.

**Features:**

- Live search functionality
- Case-insensitive filtering
- Dynamic UI updates

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository or download the ZIP.
2. Open any project folder.
3. Run the `index.html` file in a web browser.
